Verifier command

How to reproduce the current status

Clone the source/checker repository and run the repository-level verifier:

git clone https://github.com/aisknab/pnp.git
cd pnp
npm ci
npm run pnp:verify

Focused activation checks are also exposed in the source/checker repository:

npm run proof:public-theorem-activation
npm run proof:unrestricted-final-soundness-release
npm run proof:uniform-complexity-conclusion
